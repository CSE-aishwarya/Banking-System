# 🏦 SecureBank Banking Management System

## Description
Full Stack Web Based Banking System
built using Java Spring Boot and MySQL

## Technologies
- Java Spring Boot
- MySQL Database
- HTML CSS JavaScript
- JWT Security

## Features
- User Registration and Login
- Account Management
- Deposit Withdraw Transfer
- Loan Management with EMI Calculator
- PDF Statement Download
- Admin Panel
- QR Code and Chatbot

## How to Run
1. Clone repository
2. Setup MySQL database
3. Update application.properties
4. Run Spring Boot application
5. Open http://localhost:8080
